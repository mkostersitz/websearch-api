# User Management

Manage users, groups, and permissions in the WebSearch API.

## Overview

The user management system provides:
- User creation and administration
- Group-based permissions
- Role assignment
- CSV bulk import

## Creating Users

Navigate to **Users & Groups** → **Add User**

## Managing Groups

Create groups to organize users and assign policies.

## Bulk Import

Import users via CSV file upload.
