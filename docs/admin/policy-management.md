# Policy Management

Configure search policies and content filtering rules.

## Policy Types

1. **Enterprise Search Policies** - Organization-wide rules
2. **Custom Policies** - Group or user-specific rules
3. **Parental Controls** - Age-appropriate filtering

## Creating Policies

Navigate to **Policies** → **Create Policy**

## Policy Assignment

Assign policies to users, groups, or API keys.
